在您的 facebook开发者帐户中设置应用

2025年6月2日 作者 fb hub

这是您在开始之前需要了解的内容。

要求

  • Facebook广告帐户 ID
  • 平台特定细节
平台 细节
安卓
  • Google Play 软件包名称
  • 类名
iOS
  • 捆绑包 ID
  • iPhone 商店 ID
  • iPad 商店 ID

注册成为 Meta 开发者

如果您已经注册为 Meta 开发人员，请跳过此步骤。

  1. 转到开发人员的 Meta。
  2. 登录您的帐户。
  3. 选择“开始”注册为 Meta 开发人员。

在 Facebook 中创建应用程序

本节介绍如何在 Facebook 中设置应用。如果您已经设置了应用，请跳过本节。

  1. 登录您的Meta 开发者账户。
  2. 转到“我的应用程序”部分。
  3. 选择创建应用程序。
  4. 在您希望您的应用做什么下，选择您的应用的用例。
  5. 选择应用程序的类型。
  6. 提供应用程序名称、联系方式，然后选择“创建应用程序”。
    • 创建应用后，系统会分配一个 App ID。在 Adjust 中设置 Facebook 时，您需要此值。
  7. 转到“设置”>“基本”，然后选择“添加平台”。
  8. 选择您的平台并输入其他特定于平台的详细信息：
安卓
  1. 选择 Android 商店，然后选择“下一步”。
  2. 输入软件包名称和类名。Meta 会使用这些信息让尚未安装您的应用的用户从 Play 商店下载。
  3. 如果您要从 Adjust 导入事件，请关闭“自动登录应用内事件”的开关。
  4. 选择“保存更改”。
    • 保存更改后，安装引荐来源解密密钥 将被填充。
iOS
  1. 输入Bundle ID以及 iPhone 和 iPad 的Store ID。Meta会使用这些信息，让尚未安装您的应用的用户从 App Store 下载。
  2. 如果您要从 Adjust 导入事件，请关闭“自动登录应用内事件”的开关。
  3. 选择“保存更改”。
  1. 转到发布 > 上线，然后选择上线以发布您的应用。
  2. 在左侧导航窗格中的“提交审核”部分下，您需要使用Meta完成应用审核和业务验证流程。

登录您的 Meta 开发者帐户并选择我的应用。

重要价值

在 Adjust 中设置 Meta时，您需要从您的应用中获取以下值：

  • （必需）App ID – 在您的 Meta 开发者账户中，前往您的应用。记下App ID。
  • （仅限 Android – 推荐）解密密钥– 在您的 Meta 开发者账户中，前往“设置”>“基本”。在Android部分下，记下“解密密钥”。仅当您通过原始数据导出功能导出设备级数据时才需要此密钥。这可以丰富 Android 用户的广告系列数据。iOS 用户请忽略此设置。

应用程序 ID

将广告帐户 ID 与你的应用关联

在本部分中，了解如何将用于投放广告系列的广告帐户与应用关联。你可以将多个广告帐户与应用关联。

  1. 在 Meta for Developers 中，选择“应用程序”并打开您的应用程序仪表板。
  2. 转到应用程序设置>高级。
  3. 在广告帐户部分下，输入用于运行广告活动的授权广告帐户 ID 。

设置 Meta App 广告活动

在本节中，了解如何在 Facebook、Instagram 和 Facebook Audience Network 上推广您的应用。

  1. 在Meta Ads Manager中，选择与您的应用关联的广告帐户。
  2. 选择“创建”以创建新的活动。
  3. 选择应用推广作为广告活动目标。
  4. 选择适当的活动类型。
  5. 转到1 广告集，然后在应用程序部分下输入您想要推广的应用程序名称或确切的商店网址。
  6. 配置此部分中的其余设置。有关详细说明，请按照Meta 文档中的步骤操作。

设置应用事件

在本节中，了解如何从 Adjust 向 Meta 发送事件活动。

  1. 在Meta Ads Manager中，转到 Events Manager，然后选择Partner Integrations。
  2. 搜索并选择调整。
  3. 选择“连接帐户”。
  4. 选择您创建的应用程序。

您在 Meta Ads Manager 中的设置已完成。现在，您需要前往 Adjust，并将Adjust 事件映射到 Meta 事件。

当您的事件被触发时，您将能够在元事件管理器中的数据源下看到事件详细信息。

在 Meta 中禁用事件日志记录

重要的：

在 Adjust 中设置事件映射后，请在 Facebook SDK 中禁用事件日志记录。这可确保事件仅被测量一次。

您无需在 Facebook SDK 中定义事件，而是可以在 Adjust 上定义等效事件并将其链接到 Meta。在 Adjust 上定义事件还允许您将事件链接到其他平台。

要禁用 Facebook SDK 事件日志记录，请按照以下步骤操作：

  1. 登录您的Meta 开发者账户。
  2. 选择您的应用程序。
  3. 进入“设置”>“基本”。
  4. 对于每个平台，请确保已关闭“自动登录应用内事件”的开关。

此外，您还需要在应用中执行以下操作：

  1. 在 Facebook SDK 中禁用自动事件日志记录 – iOS | Android | Unity
  2. 删除 Facebook SDK 中的手动记录事件 – iOS | Android | Unity

（仅限 iOS 应用）粘贴连接 URL

您需要从 Adjust 的转化中心复制连接网址，并将其粘贴到 Meta 事件管理器中。这将为 Meta 提供指向转化价值映射的链接。

  1. 在Meta Ads Manager中，转到 Events Manager，然后选择Data sources。
  2. 选择您创建的应用程序。
  3. 前往“设置”。
  4. 在为 SKAdNetwork 配置应用事件下，选择开始设置事件或编辑事件（如果您之前已配置）。
  5. 选择从合作伙伴应用程序导入，然后选择下一步。
  6. 粘贴您从 Adjust 的转换中心复制的连接 URL ，然后选择连接。
  7. 检查导入的配置，然后选择“确认”。
