如何投放 Facebook 广告：Facebook 广告新手指南

2025年6月2日 作者 fb hub

在社交媒体上投放付费广告是扩大内容覆盖面的直接方法 — — 尤其是在 Facebook 这样的强大平台上。

凭借超过 30 亿月活跃用户，Facebook 毫无疑问是最受欢迎的广告投放社交媒体平台。

但Facebook 算法是一个难以攻克的难题，特别是对于刚刚起步的你来说——无论你是小企业主、创作者还是社交媒体经理——因此，建立目标受众的明智方法是通过 Facebook 的广告来接触他们。

因此，当谈到社交媒体策略时，了解如何在 Facebook 上做广告是必须的。

在 Facebook 上做广告有哪些好处？

精准定位目标客户

Facebook 拥有大量用户数据，营销人员可以在定位受众时使用这些数据。

创建 Facebook 广告时，您可以获取 Facebook 用户的人口统计数据，例如性别、位置和年龄。Facebook 与其他广告平台的不同之处在于，它提供基于生活方式的定位选项，例如：

  • 兴趣
  • 行为（例如最近的购买）
  • 教育和工作经历
  • 收入
  • 政治立场
  • 重大生活事件
  • 口语

如果您愿意让AI为您完成所有这些工作，那么2023年发布的Meta Advantage+功能将实时学习您的广告效果。据Meta称，这将有助于“最大限度地发挥AI在您的广告系列中的性能优势，并最大化每次广告展示的价值”。

获得强大的结果

一旦掌握了 Facebook 广告管理器，学习如何投放 Facebook 广告就很容易了，而且很快就会看到效果。

最近一项针对美国零售商的调查显示，41% 的零售商认为Facebook 广告的广告支出回报率(ROAS) 最高。相比  所有行业 8.25% 的平均转化率，这一数字可谓巨大。

在 Facebook 和 Instagram 上触达数十亿人

由于Meta同时拥有Facebook和Instagram，因此可以轻松整合这两个平台。这意味着广告可以同时在两个平台上投放，即使您尚未在 Instagram 上活跃，也能接触到更多社交用户。Facebook 和 Instagram 上的广告覆盖潜力无与伦比。

营销机构（90%）和小型企业（85%）都认为 Facebook 新闻推送广告最为有效。然而，Instagram 动态和Instagram Stories分别位列第二和第三，成为本榜单中表现最佳的元广告渠道。

实时追踪广告效果

虽然 Meta在 2021 年放弃了其完整的 Facebook Analytics 平台，但它仍然拥有丰富的资源，可用于了解广告效果，包括覆盖面、参与度、转化率等指标。这些数据会实时更新，因此您可以根据效果调整广告系列。

Facebook 当前分析功能的另一个重要优势是能够将Meta Pixel添加到您的网站。大多数内容管理平台都允许您轻松地将这段代码添加到您的网站，并跟踪 Facebook 广告的效果。您可以监控用户行为，并为未来的 Facebook 广告活动创建特定的目标受众。

投放 Facebook 广告之前

在开始之前，务必思考投放广告的原因以及目标。在投放广告之前，确定衡量成功的标准，才能确定哪个广告目标最符合您的需求。

例如，如果您想通过 Facebook 广告增加移动应用的下载量，那么成功指标可以是第一个月下载量达到 100 次。如果您追求的是潜在客户开发，那么衡量成功的标准可以是更大的电子邮件列表。在创建广告系列之前，您应该思考投放广告的目的。

如何在 Facebook 上做广告

Facebook 的所有广告活动都通过Facebook 广告管理器工具进行，因此学习如何在 Facebook 上投放广告意味着要掌握它。值得庆幸的是，它非常容易使用。我们将在本指南中逐步指导您如何在 Facebook 上投放广告。

Facebook 广告管理器入门

注册 Facebook 时，系统会默认为您分配一个个人广告账户 ID。您可以前往广告管理器，在左上角查看此 ID。

要使用 Facebook 广告管理器，您需要一种经过验证的付款方式，并且：

  1. Facebook 页面或
  2. 成为其他人主页的管理员、编辑或广告商

如果您还没有公司主页，可以按照Facebook 的步骤创建一个。您创建的任何主页都会自动获得一个广告账户。

如果您希望为其他人创建的主页做广告，则需要要求主页管理员为您分配该主页的管理员、编辑者或广告商角色。

如果您已拥有主页广告帐户，但想要创建一个新的，则需要商务管理平台帐户。注册商务管理平台后，即可创建新的广告帐户。

这些步骤适用于在台式机或笔记本电脑上使用它。

找到 Facebook 广告管理器最简单的方法是点击并收藏此链接。您将直接进入 Facebook 广告管理器，访问您的个人广告账户。如果您管理多个广告账户，请使用账户下拉菜单选择合适的账户。

要直接从您的企业主页导航到广告管理器，请前往左侧边栏，点击任意 Facebook 主页的“广告中心”下拉箭头，然后从下拉菜单中选择“所有广告” 。在该主页底部，有一个点击“广告管理器”的选项。

点击 Facebook 的广告管理器即可开始使用。您将进入 Facebook 广告帐户页面，其中会简要概述您的广告帐户。如果您可以访问多个广告帐户，请选择您想要投放广告的帐户。

如果您更喜欢使用移动设备，可以使用适用于 Apple 和 Android 的Facebook 广告管理器应用。

如何创建 Facebook 广告

这是使用最新版本的 Facebook 广告管理器在 Facebook 上投放广告的完整分步指南。

1. 创建广告

单击绿色的“创建广告”按钮开始创建您的第一个广告。

Facebook 会将您带到一个页面，提示您选择购买类型和活动目标。

2. 选择您的购买类型

Meta 的广告有两种不同的购买类型：竞价和预订（以前称为覆盖率和频率）。以下是 Meta 对这两种类型的解释：

竞价：提供更多选择、更高效、更灵活，且结果更难以预测。广告可在 Facebook、Messenger、Instagram 和 Meta Audience Network 上投放。

预订：让您提前规划和购买广告系列，获得可预测的效果目标，并更好地控制投放频率。广告可在 Facebook 和 Instagram 上投放。

两者之间的主要区别在于广告系列设置和广告系列目标。例如，竞价广告允许设置各种广告系列目标，而预订广告只有两个目标：知名度和参与度。

值得一读的是 Meta 的指导，了解哪种选项最适合您的需求。

3. 选择你的广告系列目标

Meta 和 Facebook 可让您根据最终目标设置广告。有六种广告方式可供选择：

  1. 认知度：您的广告将展示给最有可能记住您的用户。根据 Meta 的数据，此目标类型有利于提升覆盖面、品牌知名度、视频观看量和店铺位置认知度。
  2. 流量：此广告类型会将用户引导至特定目标，例如您的网站、活动或下载链接。它最适合用于链接点击、着陆页浏览和致电。
  3. 互动：这种广告类型能让您更有可能收到消息、应用内购买、点赞和评论。它也非常适合提升视频观看量、转化率和通话量。
  4. 潜在客户：您的广告可以帮助您收集潜在客户的信息。这有助于表单、转化和致电。
  5. 应用推广：为您的应用吸引更多用户或鼓励现有用户继续使用它。
  6. 销售：找到可能购买你的产品或服务的用户。这条途径显然对转化率和应用内销售非常有效。

4. 为你的广告系列选择一个名称

选择营销目标后，系统会要求您为广告系列命名。请确保所有参与广告制作的人员都能识别您的广告系列名称。

5. 声明您的广告类别

不同地区有不同的广告规则，尤其是在涉及信贷、就业、住房、社会问题、选举或政治等类别时。如果您的广告属于这些类别之一，则值得在此处声明，以确保其符合 Facebook 和各个国家/地区的广告标准。

6. 选择是否进行 A/B 测试

此时，您可以启用 A/B 测试功能，尝试不同的图片、文字、受众群体或展示位置，以最大限度地提升广告效果。完整的设置将在稍后进行，只需单击此开关即可启用此功能。

7. 选择是否启用 Advantage+ 广告系列预算

此功能仅在您选择竞价购买选项时可用，这意味着 Facebook 将自动在您的广告组中寻找最佳可用结果机会，并实时分配您的广告系列预算以获得这些结果。

Advantage+ 广告系列预算（以前称为广告系列预算优化）最适合至少有两个广告组的广告系列。

8. 缩小你的绩效目标

Facebook 现在允许您非常具体地设定广告的最终目标。第 3 步中提到的每个广告系列目标都有一组单独的具体目标——例如，认知度目标涵盖了从更广泛的“最大化覆盖面”到高度具体的“最大化两秒连续视频观看次数”等所有内容。

如果您是 Facebook 广告新手，那么在确定适合您的目标和您分享的内容的特定目标之前，值得尝试更广泛的选项。

9. 确定你的广告支出和时间表

设置广告预算时，请务必记住，此数字代表您愿意支出的最高金额。您可以设置每日预算，也可以设置整个广告系列的总预算。

  • 每日：每日预算会限制您每天在广告上的支出。Facebook 会根据您设定的条件，找到合适的时间和地点向目标受众展示您的广告。一旦达到预算上限，广告当天将停止投放。第二天将重新开始投放。
  • 终身预算：终身预算是指您在整个广告系列投放期间的最高预算。您只需向 Facebook 提供一个数字和结束日期，Facebook 就会根据广告效果支出这笔资金。一旦达到预算上限，广告系列就会结束。

10. 定制你的目标受众

Meta 最近对广告管理器中的受众自定义设置进行了一些更改，因此，如果您是普通用户，现在的情况可能看起来会有些不同！

以前，营销人员可以关注从位置到行为和兴趣的一切；现在，您只能选择三个选项：位置、年龄和语言。

但如果您想要更具体，请不要担心：新的Meta Advantage+ 受众功能可以为您完成所有广告定位。

工作原理如下：

“Meta 的人工智能使用大量信息来寻找你的受众，并在学习过程中不断发展（例如，过去的转化、像素数据以及与之前广告的互动）。

您可以依靠 Meta 的 AI 来寻找您的受众，或将您想要触达的人群的具体信息添加到“受众建议”中。Advantage+ 受众功能能够创建尽可能广泛的搜索受众，从而为 Meta 的 AI 带来极大的灵活性。

换句话说，您仍然可以提供更具体的建议（例如年龄范围、性别和兴趣）。Meta 会先定位这些受众，然后再扩展搜索范围，以便为您的广告生成最佳效果。

对于那些彻底了解目标受众并希望接触他们的 Facebook 广告管理器老手来说，这是一个好消息：您的自定义受众仍然可以在广告创建向导中使用，您甚至可以通过单击以下屏幕截图中看到的链接切换回原始受众选项。

11. 选择广告展示位置

广告展示位置决定了您的广告在目标受众群体中展示的位置。Meta 也对其展示位置工具进行了类似的改进，推出了 AI 驱动的 Advantage+ 展示位置，以帮助您最大限度地扩大广告覆盖面。

不过，您仍然可以通过选择上方屏幕截图中的“手动展示位置”选项来自行决定广告展示位置。完成后，您会看到以下选项可用：

Meta 社交平台套件中提供了多种选择：

  • 信息流：通过信息流中的广告提高您的业务知名度。
  • 故事和卷轴：通过沉浸式全屏垂直广告讲述丰富的视觉故事
  • 插播：在人们观看视频时快速吸引他们的注意力。
  • 卷轴覆盖：在人们观看短篇内容时，通过贴纸或横幅广告吸引他们。
  • 搜索结果：当人们搜索时，您的业务就会得到关注。
  • 消息：向已经与您的企业建立联系的人员发送优惠或更新信息。
  • 应用和网站：利用外部应用和网站上的广告扩大您的覆盖面

欲了解更多信息，请查看Facebook 的广告投放指南。

12. 选择你的社交账户

在下一个窗口中，您可以选择要在哪些元社交媒体配置文件上投放广告。

13. 选择广告格式

在广告设置下，您可以创建新广告、使用现有帖子或使用 Meta创意中心中的任何模型。

做出选择后，选择您的广告是以单张图片或视频、轮播（多张图片或视频帖子）还是集合（一组在移动设备上成为全屏体验的产品）的形式呈现。

14. 选择媒体并撰写文本

终于到了添加那些令人屏息的创意素材的时候了！广告管理器很可能已经提取了你在 Facebook 页面上发布的内容，但你可以通过点击广告素材框中的“编辑”下拉菜单来更改它。

上传您的照片或剪辑 — 轮播广告、视频广告，它们都在这里创建。

在此区域，如果您想为每个平台定制体验，您还可以切换 Meta 应用程序每个部分中使用的广告文案和广告创意。

在本节的下方，您可以选择适用格式的音乐（查看我们关于Instagram 音频趋势的文章，以帮助您做出选择！）

您还可以使用此部分中的各种文本选项添加您的广告文案 – 不要忘记添加适用的号召性用语 (CTA)。

15. 选择目的地

正如您现在所期望的那样，当谈到您的广告目的地时，您将拥有一系列有趣的选择 – 当人们点击您的广告时，Facebook 会将他们发送到哪里。

即时体验（以前称为 Canvas）是一种快速加载、针对移动设备进行优化的体验。它为 Facebook 提供了一种让您的受众留在其应用中的方法。换句话说，它们是您可以使用 Facebook 创建的简单、动态的落地页、表单或产品图册。

当然，您也可以选择您的网站链接，甚至添加您自己的UTM 跟踪参数。

除此之外，您还可以将人们直接发送到您在Facebook Messenger、Instagram 或 WhatsApp 中的收件箱。

16. 发布您的广告

要提交您的广告，请点击页面右下角的“发布”按钮。

广告提交后，Facebook 会保留您的帖子进行审核，以确保其符合其规范且不违反其社区准则。审核预计在 24 小时内完成。广告上线后，您会收到 Facebook 发送的确认邮件。

制作 Facebook 广告需要多少钱？

虽然具体细节比较复杂，但简单的答案是，您只需支付您想要的金额。您在上面第 9 步中设置的预算就是您在广告上的支出——然而，您从这笔支出中获得的收益可能会有很大差异。

根据Wordstream 的数据，所有行业的 Facebook 广告平均每次点击费用 (CPC) 为 0.83 美元 （相比之下，Google 广告平均每次点击费用为 4.22 美元）。

然而，深入研究特定行业，每次点击费用可能会有很大差异。例如，每次点击费用最高的行业是牙医和牙科服务，为 1.27 美元；家具，为 1.19 美元；律师和法律服务，为 1.15 美元。每次点击费用最低的行业是旅游，为 0.43 美元；艺术和娱乐，为 0.44 美元；餐饮和食品，为 0.52 美元。

制定 Facebook 广告策略

现在您已经掌握了创建 Facebook 广告的基础知识，您可以开始尝试看看哪种方法最适合您的受众。

对于上述每个步骤，停下来反思一下你的目标会很有帮助。确保你定义和创建的所有内容都符合你自己的成功标准。不确定你的受众想看什么？没问题。Facebook 有很多A/B 对比测试工具，你可以并排比较广告，找出哪些有效。发挥创意并监控广告效果，以便在必要时进行调整。
